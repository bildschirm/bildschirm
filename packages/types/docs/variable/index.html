    public DISCONNECT_REASON: Object since 1.0.0 source 

    import {DISCONNECT_REASON} from 'mission-control-client/src/index.js'

    The disconnect reason that gets passed along with the SocketEvents 'disconnect' event.

    Properties:

    NameTypeAttributeDescription
    UNKNOWN string

    An unknown disconnect reason.
    SERVER_DISCONNECT string

    The server disconnected the client. A manual reconnect would be required.
    CLIENT_DISCONNECT string

    The client disconnected from the server. A manual reconnect would be required.
    PING_TIMEOUT string

    The ping to the server timed-out. The client will automatically try to reconnect.

    public SOCKET_ERROR: Object since 1.0.0 source 

    import {SOCKET_ERROR} from 'mission-control-client/src/index.js'

    The error types that cause the SocketEvents 'error' event to fire.

    Properties:

    NameTypeAttributeDescription
    GENERAL string

    A general socket error.
    TIMEOUT string

    The ping to the server timed out.
    NO_ATTEMPTS_LEFT string

    The client ran out of attempts to reconnect to the server.
    AUTH_INVALID_TOKEN string

    The client is not granted access to the server due to the token being invalid.
    AUTH_TIMEOUT string

    The client is not granted access to the server because the client took too long to authenticate.
